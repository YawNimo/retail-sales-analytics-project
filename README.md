# Retail Sales Analytics Project

## Objective
Analyze sales performance across categories, states, and time periods.

## Tools Used
- Excel (Data Cleaning)
- MySQL (Queries)
- Pivot Tables (Quick Summary)
- Power BI (Visualization)

## Engineered Features
- Revenue
- Unit Price
- Cost
- Profit Margin

## Initial Insights
- Office Supplies generated the highest revenue.
- Florida and New York were most profitable.
- Revenue peaked in Q4.
